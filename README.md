# Asteroids-Clone-in-C-
I decided that my first coding project would be Asteroids.

List of mechanics:
1) Arrow keys for movement
2) "H" for hyperspace
3) physics based movement and drag
4) Increases asteroid count and speed in later levels
5) RNG UFO zig-zag movement
6) Collision detection for all entities
7) Main menu with retro arcade theme
8) High score file save
9) High score competition table with initial input
10) Retro sound effects
11) +1 lives every 10000 score
12) Steep difficulty curve
13) Smart spawn (Spawns large UFOs early and small UFOs late)
14) Increasingly difficult UFO ai that has lower accuracy early on an better accuracy later on

The entire game was made in C++ using SFML.
